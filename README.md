## Project Overview
This project focuses on analyzing customer characteristics and purchasing behavior for AeroFit treadmill products using descriptive statistics and probability concepts.
The objective was to identify customer profiles for different treadmill models and generate actionable business insights based on customer demographics, fitness levels, income, and usage patterns.

## Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Statistical Analysis

## Analysis Performed
- Exploratory Data Analysis (EDA)
- Customer Profiling
- Univariate Analysis
- Bivariate Analysis
- Correlation Analysis
- Outlier Detection

## Key Insights
- Identified customer segments for each treadmill model.
- Analyzed relationships between income, fitness level, usage, and product purchases.
- Observed purchasing trends based on gender and marital status.
- Evaluated the probability of customers purchasing specific treadmill products.

## Business Impact
The analysis provides insights that can help AeroFit improve customer targeting, product positioning, and marketing strategies based on customer demographics and fitness behavior.
